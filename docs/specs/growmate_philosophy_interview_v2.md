# 哲学抽出オンボーディング・インタビュー v2（選択式・スコアリング版）

GrowMate Pro 初回登録時の構造化インタビュー。
**自由記述を排し、選択（複数可）だけで「哲学の軸」と「思考タイプ」を同時に取る。** 書けない人を取りこぼさず、選ぶだけで裏でスコアが積み上がる設計。

> **P0 パイロット（クライアント合意）**: 当面は GrowMate 内オンボーディングは実装しない。本書の質問・選択肢をベースにした **専用 GPTs** で対話診断を行い、出力テキスト（800字以内・6項目テンプレ）をユーザーが事業者情報の `強み+毎回反映したい内容` 欄（DB キー `strength`）へコピペする。詳細は `docs/plans/philosophy-profile-mvp-plan.md` の「先行実装：パイロット P0」および `growmate_philosophy_system_spec.md` §0。

## 設計の4原則

1. **「あなたの哲学は？」と聞かない。** 選択を迫り、何を捨てるかを選ばせる。哲学は綺麗事ではなくトレードオフに出る。
2. **選択肢は「どれも正しいが同時には選べない」もので作る。** 模範解答が並ぶと全員同じプロファイルになり哲学が死ぬ。
3. **各選択肢に4特性スコアを仕込む。** ユーザーは選ぶだけ、裏で思考タイプが確定する。追加の診断テストは作らない（相乗り）。
4. **初版の確信度は低くていい。** 本当の精度は使用後の修正ログで上がる。

### 4特性（色）の定義
- **青＝分析**：論理・データ・根拠
- **緑＝構造**：手順・計画・再現性
- **赤＝社交**：共感・関係・人
- **黄＝概念**：ビジョン・発想・大局

※「エマジェネティクス」は商標。ツール内では診断名を名乗らず「思考特性の傾向」等の独自表現にする。

## スコアリング・ロジック

### 思考タイプ（色）
全問で選んだ選択肢の重みを色ごとに合算。
- **正規化**：複数選択で選んだ数に差が出るため、`各色スコア ÷ 全選択数` で割合化する。多く選んだ人が全色高くなる歪みを防ぐ。
- **濃い特性（dominant）**：上位1〜2色。
- **薄い特性（deficit／不足軸）**：最下位の色。← Layer 3の入力。

### 哲学の軸（クロスチェック）
- **Q3（捨てた選択）× Q4（敵）が同じ色に収束** → そのテーマを **core philosophy（確信度：高）** に昇格。
  - 例：Q3「数を追って質を落とさない（青）」× Q4「見せかけの成果だけ追う（青/黄）」→「手段の目的化を嫌う」が高確信で確定。
- **Q6（譲れない）× Q9（動機）が一致** → コア価値として確定。
- 1問にしか現れない要素 → 確信度：低で保留。

## Layer 3：不足軸レンズで「新しい哲学」を提案

`deficit`（薄い色）に対応する固定レンズで、**既存の強みを弱い軸から照射し直す**。強みは絶対に否定せず、補助線として増幅する。

| 不足軸 | 当てるレンズ（提案テンプレ） |
|---|---|
| 青が薄い | その判断、効果を数値で確かめると、力を入れる場所を絞れる |
| 緑が薄い | その成功、仕組みにすると毎回再現できる |
| 赤が薄い | その正しさ、相手の感情を通すともっと届く |
| 黄が薄い | その積み上げ、何のためかを一段上で言語化すると軸がぶれない |

- **鉄則**：「強みを捨てて弱点を埋めろ」は出させない。新哲学は常に「既存の強み × 弱いレンズ」の形でしか生成しない。
- **検証ループ**：提案は確定ではなく仮説。本人が採用／棄却／修正を選ぶ（前回の検証ループをそのまま接続）。

## 行動特性3つの回収先（修正ログ）

エマジェネティクスの行動特性（表現性・自己主張・柔軟性）はインタビューでは取りにくいので、**使用中の修正ログから推定**する。
- 表現性 ← 感情表現を足す／削る方向
- 自己主張 ← 断定を強める／和らげる方向
- 柔軟性 ← 修正の振れ幅

→ インタビューで思考4特性、使用ログで行動3特性。二段で7特性フルプロファイルが揃う。

## 完答させる仕掛け

- **1問ずつ対話的に提示。** 選択式なので所要3分。
- **Q2〜Q10は複数選択OK。** 0個は不可（最低1つ）。Q5の記述のみスキップ可。
- **「正解はない、ピンと来たものを」と毎問添える。**
- スキップが多いほどプロファイルが薄くなる旨を提示（行動者を自然にスクリーニング）。

## 実装メモ

- 初版は固定10問。動的フォロー（前回答を受けた深掘り）は v3。
- 選択肢の重みは「主要1色 +2」または「2色 +1ずつ」で統一。微調整は実運用データを見て。
- Q2の二択は業種で効きが変わるため、将来は業種タグで選択肢を出し分ける余地を残す。
- 初版プロファイルは確信度 low。修正ログ10件到達で再生成し、確信度を引き上げる二段構え。
- 「エマジェネティクス」診断名はツール表記に使わない（商標）。
